# assets/player.glb

The player's rigged 3D character model ("Vanguard"), used with baked
Idle/Walk/Run animations.

Source: three.js repo sample models
(`examples/models/gltf/Soldier.glb` in https://github.com/mrdoob/three.js),
originally created for Mixamo. It's a standard, freely-usable demo asset —
the same one used across many three.js example scenes and tutorials.
